HTML Tags and Descriptions

<h1> - <h6>
Defines HTML headings. <h1> defines the most important heading and <h6> defines the least important heading.

<p>
Defines a paragraph.

<a>
Defines a hyperlink.

<img>
Embeds an image.

<ul> and <li>
<ul> defines an unordered list, and <li> defines a list item.
